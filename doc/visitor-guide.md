# 嗨，欢迎来到这里 👋

这里是站长的个人空间，用来记录技术笔记、分享项目经验和偶尔的生活感悟。

如果你是第一次来，或者对这个博客的使用方式有疑问，这份指南可以帮到你。

## 逛一逛

### 首页

进来第一眼看到的是顶部的大图区域，上面有博客名称和站长的签名。这张背景图会根据时间自动切换——早晨、中午、傍晚、深夜各不相同，每次来可能看到不一样的风景。

往下是**文章列表**，每篇文章会显示标题、发布日期、字数、阅读时长和简介。文章多了会有分页，底部可以翻页。

页面左侧有一个**资料卡片**，里面有站长的头像、简介、快捷入口和社交图标。点「博客空间」可以看所有文章分类，点「友情链接」可以看到推荐的网站，点「关于本站」了解更多。社交图标直接跳转到站长的 GitHub、Bilibili、Twitter 等主页，邮箱图标会打开邮件客户端。

### 博客

点击导航栏的「博客」进入文章分类页。

页面上方是一个**标签云**，每个标签后面带一个数字，表示这个标签下有多少篇文章。点某个标签（比如「Vue」），就能只看 Vue 相关的文章。再点一次取消筛选。

下面是**最新文章列表**，和首页一样支持分页。

这是浏览所有文章最全的地方。

### 搜索

在导航栏点击「搜索」进入搜索页面。

页面上方有一个搜索框，输入关键词后会自动搜索（有短暂延迟，不用按回车）。搜索范围包括文章标题、简介和标签。

搜索框下方是所有标签，可以点击标签快速筛选。标签筛选和关键词搜索可以同时使用——先选一个标签，再输入关键词，结果会取交集。

搜索结果下方会显示「找到 N 篇文章」，结果多的时候同样支持分页翻页。

### 友链

导航栏的「友情链接」展示了站长推荐的网站和朋友的博客。

每个友链卡片上有头像、名字、简介和一个「访问」按钮。点击「访问」会跳转到对方的网站，会在新标签页打开，不会离开这个博客。

### 关于

「关于」页面是站长的个人介绍，内容由站长自己编写，可能是技术栈、个人经历或者想说的话。

页面底部会显示一些统计数据：今天有多少人来过、一共有多少人访问过、今天更新了几篇文章，以及你的 IP 地址（带国旗图标，根据 IP 自动判断所在国家/地区）。

## 阅读文章

点进一篇文章后，你会看到：

**顶部大图** — 如果站长上传了封面图，会在文章顶部展示。同时显示文章标题、发布日期、字数和预计阅读时间。

**文章正文** — 支持丰富的 Markdown 排版：
- 标题层级（H1-H4）
- **加粗**、*斜体*、`行内代码`
- 代码块（带语法高亮，支持多种编程语言）
- 表格
- 引用块
- 有序 / 无序列表
- 图片
- 链接

**右侧目录** — 桌面端阅读时，右侧有一个悬浮的目录栏（TOC），自动提取文章中的所有标题。点击某个标题可以快速跳转到对应位置，方便阅读长文章。目录会跟着你的阅读位置高亮当前章节。

> 在手机上看文章时，右侧目录会自动隐藏，专注阅读体验。

**底部标签** — 文章末尾会显示这篇文章的标签，点击标签可以跳转到博客分类页查看同标签的其他文章。

## 主题切换

页面右上角有一个主题切换按钮，可以在**亮色模式**和**暗色模式**之间切换。

- 亮色模式：浅色背景，适合白天或光线充足的环境
- 暗色模式：深色背景，适合夜晚或光线较暗的环境

你的选择会自动保存在浏览器中，下次访问时自动应用。首次访问时会根据你的系统设置自动选择。

## 手机上能用吗

能。整个博客做了移动端适配：

- 首页的左侧资料卡片会自动收到顶部，文章列表占满全宽
- 文章页的右侧目录自动隐藏，只保留正文
- 搜索、标签筛选、分页等操作都能正常触屏使用
- 后台管理面板的侧边栏会收起为汉堡菜单

手机和平板都能正常浏览。

## 常见问题

**页面加载很慢怎么办？**

试试强制刷新：电脑上按 Ctrl+F5（Mac 是 Cmd+Shift+R）。如果是持续很慢，可能是网络问题或者服务器那边的问题，可以稍后再来。

**文章里的代码块颜色很奇怪 / 看不清？**

试试切换一下主题。代码高亮在暗色模式下通常效果更好。如果还是有问题，可能是站长写的代码块没指定语言，显示效果会差一些。

**搜索什么都搜不到？**

确认一下关键词是否正确。搜索是匹配文章标题、简介和标签的，不会搜索正文全文。换个更短的关键词试试，比如搜「Vue」而不是「Vue 3 组合式 API 的使用方式」。

**友链里的网站打不开？**

对方网站可能暂时不可用，这不是这个博客的问题。可以过段时间再试。

**关于页显示的 IP 地址准确吗？**

基本准确，但如果你用了 VPN 或代理，显示的可能是 VPN 的出口 IP 和对应的国家/地区，不是你的真实位置。

**想转载文章怎么办？**

请先通过页面底部的社交链接联系站长，获得授权后再转载。注明出处是基本礼貌。

**我能在评论区留言吗？**

目前博客没有评论功能。如果有话想说，直接通过社交链接联系站长就好。

**这个博客用什么技术做的？**

前端 Vue 3，后端 Express 5，不需要数据库，所有数据都是文件存储。如果你也想搭一个类似的博客，可以去 GitHub 上看看这个项目。
