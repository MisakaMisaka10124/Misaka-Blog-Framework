# 2026-06-10 更新日志

## v1.3.2 - 编辑器二次更新 Bug 修复

### Bug 修复

- **修复二次更新时标题/简介/标签丢失**：文件使用 `\r\n` 换行符时，`parseFrontmatter` 的 `split('\n')` 拆行后每行残留 `\r`，导致行匹配正则 `^(\w+):\s*(.*)$` 匹配失败。表现为第二次编辑文章时 slug 变为标题、标签和简介消失
  - `parseFrontmatter` 入口处统一将 `\r\n` 转换为 `\n`
  - `handleSave` 写入前统一 `content.value` 的换行符，防止 `\r\n` 写入文件
- **修复 Vue Router 组件复用时不重新加载**：添加 `watch(slug)` 监听路由参数变化，从编辑文章 A 切换到文章 B 时正确重置表单并加载新数据
