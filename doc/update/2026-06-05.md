# 2026-06-05 更新日志

## v1.1.0 - Bug 修复与 UI 优化

### Bug 修复

- **IP 显示修复**: nginx 转发 `cf-connecting-ip` 头，后端优先读取该头获取 Cloudflare CDN 下的真实客户端 IP
- **文章更新封面丢失**: 更新文章时 frontmatter 重建保留 `cover` 字段
- **YAML 解析崩溃**: 新增 `yamlEscape` 函数，对标题/简介中的 `][`、`:`、`#`、`"` 等特殊字符自动双引号包裹转义，防止 gray-matter 解析失败
- **文章标题变 slug**: 修复 YAML 解析失败后标题回退为 slug 的问题

### UI/UX 优化

- **文章列表显示 slug**: 标题后附浅色 slug 标识，方便管理多篇文章
- **深色模式文字对比度**: `--color-text-muted` 从 `#8888a0` 提升至 `#a0a0b8`，改善可读性
- **文章列表加载状态**: 数据加载中显示「加载中...」，不再误显「暂无文章」

### 部署变更

- 无额外部署要求，nginx 默认转发 `cf-connecting-ip` 头

## v1.2.0 - 数据目录重构与后台管理

### 数据目录

- 所有可编辑数据统一存放到 `backend/data/`，前端不存储数据
- 图片迁移到 `data/images/`（avatars/backgrounds/friends/social/uploads）
- Vite 代理补充 `/images/backgrounds` 路径
- 配置文件和前端引用路径全部更新为新路径

### 后台管理

- 新增后台管理面板（/admin），包含仪表盘、文章管理、站点设置、友链、社交媒体
- 站点设置：头像上传、标题、欢迎语、个性签名、背景模式（静态/时间段/轮换）、页脚信息
- 友链管理：添加/编辑/删除友链，支持头像上传
- 社交媒体管理：添加/编辑/删除社交链接，12 个预置图标（GitHub/Bilibili/Twitter/QQ/微信/Telegram/Discord/YouTube/Steam/Instagram/微博/邮箱）
- 文章编辑器：Code/Preview 双模式，标签自动补全，封面图，图片压缩上传

### 标签系统

- 修复标签类型不一致问题（强制转为字符串数组）
- 修复 PUT 编辑文章时不同步标签索引
- 标签值在 YAML 中加引号存储，防止解析歧义

### UI 修复

- 深色模式改为纯黑色调，去掉蓝色
- 浅色模式改为纯白色调
- 修复导航栏首页蓝条始终显示（添加 exact 匹配）
- 修复后台弹窗不显示鼠标（自定义光标限制在前台布局内）
- 后台弹窗背景改为不透明
- 头像修改后加时间戳防止浏览器缓存
