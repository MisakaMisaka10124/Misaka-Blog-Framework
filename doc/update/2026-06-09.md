# 2026-06-09 更新日志

## v1.3.0 - 自动化部署

### 部署脚本

- 新增 `deploy.sh` 一键部署脚本
- 支持两种部署模式：Nginx + pm2、Docker
- 自动检测并安装缺失依赖（pm2、Nginx）
- 首次安装与版本更新自动识别
- 更新时自动备份并恢复用户数据（配置、文章、图片）
- 支持指定版本部署：`--version v1.2.0`
- 支持自定义安装路径：`--path /opt/blog`

### Nginx 自动配置

- 脚本询问用户是否自动配置 Nginx
- 自动安装 Nginx（如果未安装）
- 询问域名并生成对应配置
- 支持 HTTP/HTTPS 配置选择
- 自动启用站点并重启 Nginx

### pm2 配置

- 新增 `ecosystem.config.js` pm2 配置模板
- 配置自动重启、日志输出、内存限制
- 支持开机自启

### GitHub Actions

- 新增 `.github/workflows/release.yml` 自动构建工作流
- 推送 tag（v*）自动触发构建
- 自动打包 `dist.tar.gz` 和 `backend.tar.gz`
- 自动创建 GitHub Release

### 文档

- 新增 `doc/deploy.md` 部署指南
  - 环境要求与依赖安装（Ubuntu/CentOS/Arch/macOS）
  - Nginx + pm2 部署流程
  - Docker 部署流程
  - 数据保护策略说明
  - 故障排查与备份恢复
- 新增 `doc/update/index.md` 更新日志索引
- 更新 `README.md`
  - 添加生产部署快速开始
  - 添加更新日志索引
  - 说明 `.example` 文件使用方式

### 其他

- 更新 `.gitignore` 添加 `logs/` 目录
