# 2026-06-02 更新日志

## v1.0.0 - 初始版本

### 基础架构

- Vue 3 + Vite + TypeScript 前端
- Express 5 + TypeScript 后端
- JWT 认证 + svg-captcha 验证码
- Swagger API 文档

### 文章管理

- Markdown 文件上传，哈希 Slug 自动生成
- 文章元数据自动提取（标题、简介、标签、日期、字数）
- 文章在线编辑（Code/Preview 双模式切换）
- 文章删除（含图片目录清理）
- 文章索引自动构建和增量更新

### 图片管理

- 客户端压缩（Canvas API，最大 1920px，JPEG 82%）
- 按 slug 分目录存储（`images/{slug}/`）
- 编辑器图片粘贴和按钮上传
- `_temp` URL 自动替换为真实 slug URL
- 封面图上传并写入 frontmatter

### 标签系统

- 独立 JSON 文件存储（`config/tags/`）
- 启动时自动重建标签索引
- 文章 CRUD 时增量同步标签
- 编辑器标签自动补全

### 搜索功能

- 标题、简介、标签关键字搜索
- 标签精确筛选
- 搜索结果分页
- 300ms 防抖

### 页面

- 首页：左栏个人信息 + 右栏文章列表（分页）
- 博客分类页：标签云 + 文章列表（分页）
- 文章详情页：Markdown 渲染 + 代码高亮 + TOC 侧边栏
- 搜索页：搜索框 + 标签筛选 + 结果列表（分页）
- 文章管理页：文章列表 + Code/Preview 编辑器
- 登录页：用户名/密码/验证码
- 友链页、关于页
- 404 页面（兜底路由）

### UI/UX

- 玻璃拟态设计系统
- 深色/浅色模式切换
- 浅色模式文字对比度优化
- 自定义光标跟随
- 响应式布局（桌面端/移动端）
- 多语言切换（中/EN/繁）

### 文档

## v1.1.0 - 安全与统计

### API 安全

- 数据接口 IP 级限流（express-rate-limit），可配置窗口和最大请求数
- 共享 JWT 认证中间件（middleware/auth.ts）
- Axios 请求拦截器自动注入 JWT Token
- Axios 响应拦截器 401 自动清除 Token 并跳转登录页
- 前端路由守卫 JWT 过期校验

### 访客统计

- 访客统计服务（services/visitor.ts），JSON 文件存储
- 同一 IP 同一天只计一次，30 天自动清理旧记录
- IP 地理定位（ip-api.com），返回国家代码用于国旗显示
- About 页面展示：今日活跃、今日更新、总访客、今日访客、当前 IP + 国旗
- flag-icons CDN 引入（jsdelivr）

### 图片管理优化

- 上传文件名空格自动替换为连字符，避免 markdown 解析截断 URL
- 编辑器图片插入时 URL 自动 encodeURI 编码
- 已有文章图片文件名修复

### 文章排序

- 同日期文章按文件修改时间降序排列（新的在前）

### 生产部署

- Express 自动检测 dist/ 目录并提供前端静态文件服务
- SPA 回退路由（Express 5 语法 `/{*path}`）
- Nginx 反向代理配置模板（nginx.conf）
- nodemon 后端热重载（npm run server:watch）
- trust proxy 设置（兼容 nginx 转发和 express-rate-limit）

### 404 页面

- 新增 NotFound.vue，未匹配路由自动显示
- 玻璃拟态卡片 + 大号 404 + 返回首页按钮

### Bug 修复

- Express 5 兼容：`*` 改为 `/{*path}` 语法
- v-html 内 SVG 样式：验证码样式移至非 scoped style
- 浅色模式文字对比度优化（--color-text-muted 提升至 #5a5a6a）
- 邮箱社交链接添加 mailto: 前缀
- 首页文章排序修复（同日期按 mtime 排序）
