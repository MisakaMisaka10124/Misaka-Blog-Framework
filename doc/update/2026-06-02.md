## 技术问题修复

### 验证码显示问题
- **问题**: 验证码图片溢出容器
- **原因**: SVG 未设置宽度限制
- **修复**: 添加 `width: 120px` 容器和 `max-width: 120px` SVG 限制

### 入口链接间距问题
- **问题**: 分割线消失，文字间距过大
- **原因**: 使用 `gap` 替代了 `border-right`
- **修复**: 恢复 `flex: 1` 和 `border-right` 分割线

### 标签同步问题
- **问题**: 启动时标签索引为空
- **原因**: `buildIndex` 不同步标签
- **修复**: 添加 `syncAllTags` 函数，启动时重建标签索引

### 文章编辑标签问题
- **问题**: 编辑文章时标签不更新
- **原因**: PUT 端点不同步标签
- **修复**: 添加旧标签读取和增量同步逻辑

### 图片路径问题
- **问题**: 图片 URL 不包含月度子目录
- **原因**: 返回 URL 时未计算相对路径
- **修复**: 使用 `path.relative` 计算正确路径
