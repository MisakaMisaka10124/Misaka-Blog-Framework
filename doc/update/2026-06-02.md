# 2026-06-02 更新日志

## v1.0.0 - 初始版本

### 基础架构

- Vue 3 + Vite + TypeScript 前端
- Express 5 + TypeScript 后端
- JWT 认证 + svg-captcha 验证码
- Swagger API 文档

### 文章管理

- Markdown 文件上传，哈希 Slug 自动生成
- 文章元数据自动提取（标题、简介、标签、日期、字数）
- 文章在线编辑（Code/Preview 双模式切换）
- 文章删除（含图片目录清理）
- 文章索引自动构建和增量更新

### 图片管理

- 客户端压缩（Canvas API，最大 1920px，JPEG 82%）
- 按 slug 分目录存储（`images/{slug}/`）
- 编辑器图片粘贴和按钮上传
- `_temp` URL 自动替换为真实 slug URL
- 封面图上传并写入 frontmatter

### 标签系统

- 独立 JSON 文件存储（`config/tags/`）
- 启动时自动重建标签索引
- 文章 CRUD 时增量同步标签
- 编辑器标签自动补全

### 搜索功能

- 标题、简介、标签关键字搜索
- 标签精确筛选
- 搜索结果分页
- 300ms 防抖

### 页面

- 首页：左栏个人信息 + 右栏文章列表（分页）
- 博客分类页：标签云 + 文章列表（分页）
- 文章详情页：Markdown 渲染 + 代码高亮 + TOC 侧边栏
- 搜索页：搜索框 + 标签筛选 + 结果列表（分页）
- 文章管理页：文章列表 + Code/Preview 编辑器
- 登录页：用户名/密码/验证码
- 友链页、关于页
- 404 页面（兜底路由）

### UI/UX

- 玻璃拟态设计系统
- 深色/浅色模式切换
- 浅色模式文字对比度优化
- 自定义光标跟随
- 响应式布局（桌面端/移动端）
- 多语言切换（中/EN/繁）
