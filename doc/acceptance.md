## 项目概述

个人博客系统，基于 Vue 3 + Express + TypeScript 构建。

## 技术指标

### 前端

- **构建产物**: 约 147KB (gzip 约 56KB)
- **构建时间**: < 1s
- **TypeScript 错误**: 0

### 后端

- **启动时间**: < 2s
- **API 响应**: < 100ms

## 已知限制

1. 文件系统存储，不适合大规模数据
2. 无并发控制
3. 无缓存层
4. 无自动备份
5. 仅支持单用户
