# Misaka Blog Framework 使用指南

---

## 这是什么

Misaka Blog Framework 是一个个人博客系统。你可以在上面发布文章、展示友链、管理社交链接，访客可以浏览和搜索你的内容。

它提供了后台管理面板，所有操作通过网页完成，不需要手动编辑文件。

**主要功能：**

- 📝 Markdown 文章发布与管理
- 🏷️ 标签分类与搜索
- 🔗 友情链接管理
- 🌐 社交链接展示（GitHub、Bilibili、Twitter 等 12 个平台）
- 🌍 多语言支持（简体中文、繁体中文、英语）
- 🌓 亮色 / 暗色主题切换
- 📊 访客统计
- 🖼️ 图片上传与自动压缩

---

## 页面总览

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | `/` | 展示文章列表、个人简介、社交链接 |
| 博客 | `/categories` | 按标签浏览所有文章 |
| 搜索 | `/search` | 关键词和标签搜索 |
| 友链 | `/friends` | 展示友情链接 |
| 关于 | `/about` | 个人介绍页面 |
| 登录 | `/login` | 管理员登录入口 |
| 后台 | `/admin` | 管理面板（需登录） |

访客看到的是首页、博客、搜索、友链、关于这几个页面。登录和后台只有你自己使用。

---

## 登录后台

1. 访问 `/login`
2. 输入用户名和密码
3. 输入图形验证码（图片中显示的字符）
4. 点击登录

登录成功后会跳转到后台仪表盘。登录状态保持 24 小时，过期后需要重新登录。

## 后台功能

### 仪表盘

后台首页展示四项数据：

- **文章总数** — 你发布了多少篇文章
- **标签总数** — 一共使用了多少个标签
- **总访客数** — 累计有多少人访问过
- **今日访客** — 今天有多少人访问

下方有快捷操作按钮和最近发布的文章列表。

### 写文章

**发布新文章：**

1. 进入后台 → 文章 → 新建文章
2. 上传一个 `.md` 文件（Markdown 格式的文本文件）
3. 可选：上传一张封面图
4. 点击提交

文章会自动生成一个唯一编号（slug），出现在首页和博客页。

**编辑文章：**

1. 进入后台 → 文章
2. 在文章列表中找到要修改的文章
3. 点击编辑，可以替换 Markdown 文件或封面图
4. 保存后立即生效

**删除文章：**

在文章列表中点击删除按钮，文章和它的图片会被一起删除。

### 写 Markdown

Markdown 是一种简单的标记语言，用符号表示格式。常用语法：

```markdown
# 一级标题
## 二级标题
### 三级标题

**加粗文字**
*斜体文字*
`行内代码`

- 无序列表
- 第二项

1. 有序列表
2. 第二项

> 引用文字

[链接文字](https://example.com)
![图片描述](图片地址)

| 表头1 | 表头2 |
|-------|-------|
| 内容1 | 内容2 |

代码块：
​```javascript
console.log('hello')
​```
```

你也可以在文章开头加上元信息：

```markdown
---
title: 我的第一篇文章
date: 2026-06-06
tags:
  - 日记
  - 技术
description: 这篇文章讲的是……
---

正文内容从这里开始。
```

`title` 是标题，`date` 是日期，`tags` 是分类标签，`description` 是简介。这些都不是必填的，不写也能正常发布。

### 管理标签

标签在文章的元信息中定义，不需要单独创建。

当你在文章里写了 `tags: [Vue, 前端]`，系统会自动把这篇文章归类到「Vue」和「前端」两个标签下。访客可以在博客页按标签筛选文章，也可以在搜索时按标签查找。

### 站点设置

后台 → 设置页面可以修改以下内容：

| 项目 | 说明 |
|------|------|
| 头像 | 你的个人头像，显示在首页侧边栏 |
| 站点标题 | 网页左上角显示的名字 |
| 欢迎语 | 首页显示的欢迎文字 |
| 个人简介 | 首页侧边栏的简短介绍 |
| 关于页内容 | 「关于」页面的详细内容，支持 HTML |
| 首页副标题 | 首页大图区域的个性签名 |
| 背景图片 | 首页顶部的背景图，最多上传 6 张 |
| 背景模式 | 背景图的切换方式（见下文） |
| 页脚版权 | 页面底部的版权信息 |
| ICP 备案号 | 中国大陆备案号，没有可留空 |

**背景模式说明：**

- **固定模式** — 始终显示同一张背景图
- **按时段切换** — 根据时间自动更换（早晨、中午、傍晚、深夜各一张）
- **定时轮换** — 按设定的时间间隔自动轮换所有背景图

修改设置后，所有语言版本会自动同步更新。

### 友链管理

后台 → 友链页面管理友情链接。

**添加友链：**

1. 点击添加按钮
2. 填写朋友的名字、简介、网站地址
3. 上传一张头像图片
4. 保存

**编辑 / 删除：**

在友链列表中操作即可。

友链会显示在首页侧边栏和友链页面。

### 社交链接

后台 → 社交页面管理你的社交平台链接，最多添加 4 个。

系统内置了 12 个平台图标：GitHub、Bilibili、Twitter/X、邮箱、QQ、微信、Telegram、Discord、YouTube、Steam、Instagram、微博。

添加时选择平台、填写链接地址即可，图标会自动显示。

社交链接显示在首页侧边栏和页脚。

### 修改密码

后台 → 账户页面可以：

- 修改用户名
- 修改密码（需要输入当前密码和验证码）

## 访客看到的样子

**首页** 顶部是一张大背景图，上面显示站点标题和你的个性签名。往下是文章列表，左侧有你的头像、简介和社交链接。

**博客页** 展示所有文章，可以按标签筛选。

**搜索页** 支持输入关键词搜索文章内容，也可以按标签查找。

**友链页** 展示所有友情链接，每个友链有头像、名字、简介和链接。

**关于页** 展示你设置的个人介绍内容。

访客可以在页面右上角切换亮色 / 暗色主题。

## 常见问题

**Q: 忘记密码了怎么办？**

联系帮你部署的人，或者直接修改服务器上 `backend/data/config/core_server_config.json` 文件中的密码字段，重启服务即可。

**Q: 文章上传后看不到？**

确认文件是 `.md` 格式。如果还是看不到，进入后台仪表盘，点击「重建索引」按钮刷新文章列表。

**Q: 图片上传失败？**

支持 JPG、PNG、GIF、SVG、WebP 格式。上传前会自动压缩，一般不会有问题。如果持续失败，检查图片文件是否损坏。

**Q: 修改设置后前端没变化？**

刷新页面（Ctrl+F5 强制刷新）。如果还没变化，清除浏览器缓存后重试。

**Q: 多语言是怎么切换的？**

访客首次访问时，系统根据浏览器语言自动选择。之后可以在页面上手动切换，选择会记住。

**Q: 访客统计准确吗？**

统计基于 IP 地址，同一天内同一 IP 只算一次。所以是独立访客数，不是页面浏览量。

**Q: 怎么备份？**

让管理员备份服务器上的 `backend/data/` 文件夹，里面包含所有配置、文章和图片。

**Q: 能在手机上用后台吗？**

可以。后台管理面板做了响应式适配，手机上侧边栏会收起为汉堡菜单，所有功能都能正常使用。
